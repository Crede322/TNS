# Typescript Networked Store

Интернет-магазин центральных процессоров (некоммерческое веб-приложение)

## Стек :

- React
- Typescript
- Redux Toolkit
- Supabase
- Swiper.js React
- React Router
- React-hook-form
- CRA, node.js v.10.8.0
